What Is SentinelPay?

SentinelPay by AfroSec Analytics is an AI-powered fraud detection system that analyzes financial transactions in real-time. Unlike traditional rule-based methods, SentinelPay adapts dynamically to detect evolving fraud patterns using machine learning. It offers scalable, cost-effective protection by integrating via REST API and deploying with Docker. Through a user-friendly web interface, it delivers instant fraud alerts, data insights, and performance tracking.

The Importance of Anomaly Detection

Fraud tactics are growing more sophisticated—identity theft, synthetic accounts, phishing, and unauthorized transfers are just a few examples. Traditional fraud systems generate high false positives and lag behind emerging threats. Real-time anomaly detection empowers institutions to respond instantly, reducing financial losses and operational strain while protecting consumers.

How our Model Classifies Transaction Anomalies?

SentinelPay uses a three-layer architecture:

  • Preprocessing Layer: Cleans and prepares transaction data by encoding transaction types, scaling features, and removing irrelevant fields.
  • Model Layer: Trains and evaluates a DecisionTreeClassifier using labeled data to distinguish normal from fraudulent behavior.
  • Interface Layer: Lets users upload transaction files, view model predictions, and monitor flagged anomalies via a responsive web dashboard.

Once the model processes the data, it predicts whether each transaction is fraudulent or not, triggering alerts and generating real-time insights for immediate action.
